
| Command                              | When to use             | Description                                        |
| ------------------------------------ | ----------------------- | -------------------------------------------------- |
| `npx prisma db push`                 | Dev only                | Sync schema directly to DB (no migration tracking) |
| `npx prisma migrate dev --name init` | Dev or first-time setup | Create migration + sync                            |
| `npx prisma migrate deploy`          | Prod                    | Apply migrations only (Render startup scripts)     |
| `npx prisma generate`                | Anytime schema changes  | Regenerate Prisma client and Zod                   |
| `npx prisma studio`                  | Dev debugging           | View/edit DB                                       |
| `npx prisma db pull`                 | Reverse sync            | Pull schema from existing DB                       |
| `npx prisma format`                  | Code hygiene            | Format schema.prisma                               |
